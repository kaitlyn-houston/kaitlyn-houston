Backup & restore

Everything you log lives only in this browser. Export a backup file regularly, and keep it somewhere safe so you can restore it after clearing browser data or switching devices.
Includes: food log, goals, favorites, weight history, and imported Garmin data. Excludes: your USDA API key and Google Calendar sign-in, for safety — you'll re-enter those after a restore.

Account & sync

Not set up.
This needs a free Firebase project of your own — Google's backend service — so your data can follow you between devices. In the Firebase console: create a project, enable Email/Password sign-in under Authentication, create a Firestore database, then copy your web app's config object from Project settings and paste it below as JSON.
Everything syncs except meal photos, which stay on the device you took them on — syncing them would need a paid Firebase plan.

Reminders

These check while the app is open and show a banner at the top. If you allow notifications, they'll also try to alert you directly — but iOS limits background notifications for web apps, so they work best while you have the app open or have opened it recently.
